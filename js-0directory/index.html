发表于 2016-05-25   |   分类于 爽爆天系列 ， JavaScript基础   |     |  

爽爆天移动开发之 - 《JavaScript基础》 是一个系列教程。

此系列教程的目的是：

  1. 普及 JavaScript基础，将自己的实践经验共享出来，为ReactNative教程提供基础。
  2. 复用已有的开发经验，提高移动开发业内的整体水平，加快大家的开发速度。
  3. 结交更多的技术大牛，大家一起学习、互助、提高。

每一章遵循这样的结构安排（下面的XX代表每章要讲的内容）：

  1. 是什么：XX是什么，对XX进行概述介绍。
  2. 为什么：XX是为了解决什么问题，解决这个问题有些什么现有的方法，为什么要用XX。
  3. 怎么样：XX是怎样工作来解决问题的。
  4. 还能怎样：进阶应用，怎样优化XX，还有哪些延伸的应用。
  5. 操作笔记：命令、代码、步骤记录。
