移动app应用架构总结

发表于 2016-04-28   |   分类于 项目管理   |  

移动app应用架构

框架总架构

层 功能描述
subModule 模块内分子功能
module 应用内分模块
core 与应用相关基础库
base 与应用上层无关基础库

base

基础框架层

此层里放的都是互不相干的基础库，主要包括：

层类别 功能点
网络 network 应用内分模块
数据层处理 model 数据模型库、数据缓存
数据编码 encode 加解密
设备信息 device 屏幕、软件版本等
文件管理 filemanager
图片处理 image

core

应用核心层

此层里放与应用相关联的基础库，主要包括：

层类别 功能点
用户统计 statistic 一般统计、crash统计
push
分享 share
view控件 views

module

层类别 功能点
busniss 模块对外对内暴露的业务/数据模型（对外提供的数据模型）
subModules 各个子功能

subModules

层类别 功能点
controller 视图控制器，组合子功能业务
view 视图
viewModel 对视图提供数据支持的模型/request类
model 完成内部逻辑需要的数据模型
